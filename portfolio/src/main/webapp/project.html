Database System for E-Commerce

Spring 2020

In a team of two, we implemented a database system on Google Cloud for an e-commerce called Red Bubble, with PostgreSQL as a class project. Then, we programmed a functional website with Flask to interact with the database, such as displaying products in the catalog, verifying user logins, and posting product reviews.

Web Server with TCP Protocol

Spring 2020

I programmed a web server from scratch in C that allows up to 5 client connections. This server has two main pages: the home page and a form. The home page is a html file with images. The form is a simple search box where the program searches through the back-end database for the user entered keyword. The search results are output to client in a table format.
